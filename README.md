# # 🌱 Raízes do Futuro

Projeto educativo interativo sobre **Agricultura Regenerativa**, desenvolvido para conscientizar sobre a importância da conservação do solo, sustentabilidade no campo e produção agrícola responsável.

## 📚 Sobre o Projeto

O site apresenta informações, desafios interativos e soluções sustentáveis relacionadas à recuperação da fertilidade do solo e à agricultura regenerativa.

## ✨ Funcionalidades

- 📊 Dashboard com indicadores ambientais
- 🌾 Diagnóstico dos principais problemas do solo
- 🌱 Simulador de recuperação do solo
- 🎯 Quiz interativo com pontuação
- 🏆 Certificado de participação
- 📖 Perguntas frequentes (FAQ)
- 🎞️ Carrossel de casos de sucesso
- 🎥 Vídeo educativo
- 🌙 Modo escuro
- ♿ Recursos de acessibilidade
- 📱 Design responsivo

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts (Poppins)

## 🎯 Objetivo

Promover a educação ambiental e demonstrar como práticas agrícolas sustentáveis podem recuperar ecossistemas, aumentar a produtividade e garantir a segurança alimentar das futuras gerações.
